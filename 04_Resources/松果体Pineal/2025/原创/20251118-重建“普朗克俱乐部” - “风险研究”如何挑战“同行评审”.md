

> 投出下一个诺贝尔奖。

“政治只属于当下，而方程式属于永恒。”

这是我在The Time Project这篇文章中十分有感触的一句话，人类遗忘那些站上政治或者经济巅峰的人的速度远超你的想象，但是人们对于那些真正推动人类文明进步的科学家却保有长久的记忆。

我们应当投资科学。这不仅是对人类进步的根本投资，也为我们此生提供了一个值得追寻的崇高使命。

然而，当前的科学资助体系正遭遇严重瓶颈。一方面，资助机构承受着对资金使用和效率负责的巨大压力，这本应是对纳税人负责的体现。另一方面，随着科研范式的演变和大学的急剧扩招，同行评审制度被大规模应用。

这一体系在建立之初，确实带来了科技的迅猛发展。但运行几十年后的今天，同行评审制度及其背后的官僚化资助机构，反而日益成为科学进步的桎梏。体系内的技术官僚倾向于规避风险，不愿资助那些可能颠覆常识的猜想和实验，转而青睐那些短期内成果可见、确定性高的“安全”项目。

诚然，我们不否认这些短期项目仍能推动技术应用和渐进式进步，但它们在很大程度上是建立在过去重大科学突破的“红利”之上。这就像我们手机中的各项技术，其根基是过往的数十项诺贝尔奖级别的原始发现。

正是过去层出不穷的原始创新，为我们积累了一个丰厚的科学与技术储备池。然而，这个创新的源泉如今正濒临干涸。倘若我们继续墨守成规，总有一天，当科学家们再去探寻颠覆性的原始发现时，将只会发现源头已然枯竭，再也无法得到任何回应。

引用《Scientific Freedom》书中的比喻：这是悬挂在当今科学界所有人头上的达摩克里斯之剑。我们不知它何时落下，但**科学突破日渐乏力**的现状似乎在警示我们——这一刻已**为时不远**。

面对这种停滞，一种流行的解释是：科学之树上“低垂的果实”已被摘完，如今的突破需要更长久的努力。但事实是，“科学并不缺乏机会，很少有领域是被完全理解的”。

Don Braben的风险研究（Venture Research）的团队在10年间资助了约40项提议，总耗资不到2000万英镑。但其成果丰硕：**其中至少14项从根本上改变了我们对重要问题的思考方式**，并**间接催生了1项诺贝尔奖、开辟了多个全新领域、孵化出一家市值数亿英镑的公司**。无论从投资效率还是文明贡献衡量，这都是惊人的成就。

形成鲜明对比的是英国政府的资助方案。2005年，英格兰高等教育资助委员会向英国各大学提供了总额12亿英镑的“与研究质量相关”的资金。这些资金下放给大学，由其按自己的优先级自由使用，**结果却未能催生任何一项真正脱离传统范式的成果**。

看似“高效”的同行评审与问责体制，实际上正造成巨大的资源错配与浪费。那些出于好意（对纳税人负责）的变革，最终只创造出臃肿笨拙的官僚机构，系统性地抑制了主流之外的探索和对传统的挑战。讽刺的是，这类资助方案在各国政府中屡见不鲜，文件上甚至明确要求资助“反共识”、“具突破潜力”的项目，但现实却是无一例外都收效甚微。

因此，科学的潜力几乎与一百年前一样巨大，问题出在我们的体制。只有恢复激发科学潜力的自由，我们才能创造一个二十一世纪的“普朗克俱乐部”，并收获那些意想不到的突破。正如普朗克所言：“新的科学真理不会通过说服反对者而获胜，而是反对者最终会死去，熟悉它的新一代会成长起来。”

而僵化的同行评审制度，无疑极大延长了新一代科研人员的等待时间。当他们终于熬到有资格获取资助时，往往已垂垂老矣，磨灭了年轻时的好奇心与探索热情，更没有体力和勇气去颠覆自己毕生的学术成果。此外，“获得资格”远不等于“获得资助”。即便是Venture Research的案例中，也不乏学院院长试图为颠覆性的研究方向申请主流政府资助，但绝大多数仍遭拒绝。连功成名就的科学家都要耗费九成精力去获取资金和应对官僚主义，这多么令人痛心。

那么，Venture Research如何筛选那些真正值得被投资的项目和科研人员呢？

答案出乎意料地简单，即将看事变为看人。

Don Braben本人在启动项目时已脱离科研一线许久，且资助的领域（如生物学、化学）大多非其所长。但他发现，那些（成功的）科研人员身上所带有的品质和特性却**高度一致**。总结《Scientific Freedom》一书，筛选条件如下：

1. 颠覆性潜力：候选人必须属于极少数“能够彻底挑战现有知识框架、带来范式转移”的科研人员。我个人的理解是那些潜在的诺贝尔奖得主，他们意图颠覆当前的科学研究共识甚至常识。
    
2. 卓越但是高风险：遴选时不要求已发表高影响论文，也不要迎合主流评价指标；相反，它主动避开同行评议与影响因子，去寻找那些因观点过于前沿而可能被常规体系误判的人才。在只有10%-20%的研究能够获取资助的情况下，这样的人才可以说不在少数。
    
3. 可识别的稀有度：入选者的数量级要达到“每十年全国范围内不超过几位”的级别。我认为这是符合正态分布定律的理性之举。
    

一并总结Venture Research的筛选流程如下：

1. 每所参与大学每年推荐 1–2 名候选人。
    
2. 由退出活跃研究的资深专家组成遴选小组进行秘密评审，重点在于询问候选人的科学研究想法、研究兴趣以及下一步行动。完全不询问该研究的实际价值和作用。
    
3. 评审完全独立于常规经费竞争流程。申请者可以被邀请与选拔者非正式地讨论他们的想法，从而避免所有官僚程序。
    
4. 最终由BP（英国石油公司，Venture Research的资助方）董事会确定是否资助，由Don Braben以及那位退出活跃研究的资深专家负责游说，大多数情况下董事会都会同意。
    

当资助决定后，科研人员会立即得到下面的支持：

1. 直接财务支持：提供“足够让他们三年内无后顾之忧”的经费，真正实现“完全的学术自由”。这是一个看似是资金无底洞的条件，但是真正对科学具有好奇心的人反而会克制花费得之不易的研究资金。
    
2. 组织会议与交流：定期举办跨学科“科学盛宴”，演讲与讨论时间各占一半，帮助 grantees 互相激发新思路；此外组织自发形成的研讨会（例如“coherent systems”“theoretical engineering”等）让 researchers 以开放心态讨论大问题。
    
3. 促进合作与技术转移：通过会议和工作组促成非正式、富有成效的合作项目。
    
4. 如果三年资助研究仅取得阶段性成果，可以由研究人员自由选择是否继续开展该研究，从而获得下一个三年的资金支持。
    

“我们其实并不真正理解什么是创造力。在这种情况下，我们所能做的最多，就是给予那些具备创造性思维能力的人足够的自由，让他们能够将自己的想法付诸实践。”我想Don Braben的这句话足以概括Venture Research的核心理念。

《Scientific Freedom》的末尾记录了Venture Research十年间资助的所有项目，如果你想要了解更多细节，欢迎阅读完下面的播客稿后再阅读原书，我相信你会有非常大的收获。

在这次对话中，我与 Donald Braben 讨论了他的“风险研究”（venture research）倡议、同行评审（peer review），以及如何培养出21世纪的马克斯·普朗克（Max Planck）。

几十年来，Donald 一直是改革我们资助和评估研究方式的坚定倡导者。从1980年到1990年，他负责BP的“风险研究”项目，在那里他有机会将自己的想法付诸实践。考虑到该项目每年花费约200万英镑，却促成了至少一个诺贝尔奖和一家市值数亿美元（centi-million dollar）的公司，我会说这个项目是成功的。尽管如此，它还是在1990年被关闭了。

我们的大部分对话都重点围绕他的书《Scientific Freedom》（科学自由），这本书刚由 Stripe Press出版。我猜如果你在听这个播客，你也会喜欢这本书。那么，接下来就是我与 Donald Braben 的对话。

**主持人：** 您能用自己的话解释一下“普朗克俱乐部”的概念吗？

**主持人：** 是的。

**Donald Braben：** 所以我……我坚信21世纪有可能出现“普朗克俱乐部”，这是支撑我前进的动力。这是可能的，但**眼下**它不会发生。

我的意思是，关于同行评审已经有堆积如山的文献了，绝对是海量的。但其中大部分似乎都是由那些至少倾向于维持现状 的人写的。所以他们得出的结论是：同行评审很棒，也许只在跨学科研究（multidisciplinary research）上有点问题，那可能会引发一些麻烦。

这是（科研）当权派（establishment）的观点。所以他们采取措施，试图在_仍然使用_同行评审的前提下，稍微简化跨学科研究的流程。

因此，当传统……当资助机构说“跨学科研究有问题”时，他们实际上是在说“风险研究有问题”（venture reserves，_应为venture research_）。是的。所以，在这个问题 解决之前，我们不会有（新的）“普朗克俱乐部”。当然，我在书里提出了解决方案。

**主持人：** 是的，没错。所以我想，关于这本书，我实际上把它看作是一份制作精良、文笔雄辩的政策提案。就像，我觉得你完全可以把这本书交给政策制定者说：“照这个做。” 我猜你（可能）……但显然没人这么做，对吧？

**主持人：** 您真的去找过政府机构，甚至亿万富翁吗？您谈论的金额（每年200万英镑）小得惊人。为什么人们不愿这么做呢？他们是怎么回应的？

**Donald Braben：** 帕特里克·科里森（Patrick Collison，Stripe公司CEO）是唯一回应过的亿万富翁。我见过大概六七个亿万富翁，他们都想……他们都想用**自己的**方式做事。你知道，他们都想……这很公平，这很公平。他们都想用自己的眼光来看待大学。

他们无法睁开眼睛，放下身段倾听科学家真正想做什么。而要了解科学家真正想做什么，你不能……你不能直接问他们。你必须和他们交谈，聊很久，他们才会透露自己内心真正想做的事。

我相信原因大致如下：大约在1970年，出于政治原因而非科学原因，大学经历了一次大扩张，（持续了）二三十年。大学和学者数量激增。我真的是指，增长了三到四倍，具体取决于国家。真的非常庞大。

因此，那个或多或少保障了每个人自由的旧系统（我倡导的“人人享有自由”的权利）无法维持了。

所以，我们现在所做的，是为筛选“风险研究者”制定了绝对的筛选规则。这些规则……你知道，是花了一些时间才开发出来的，但它们行之有效。并且为世界打开了全新的视角，看待世界的全新方式。

**主持人：** 是的，看，我的意思是，您在BP的往绩记录看起来非常好，对吧？您促成了一些本不可能发生的研究，并间接带来了诺贝尔奖。对。我不知道还能拿出什么比这更有力的证据来证明您的方法有效。所以

**Donald Braben：** 嗯，嗯，这么多年来，你看，那些为资助机构工作的科学家，他们一直在建议政客如何“配给”（ration）研究经费而不影响科研。他们提出的方法就是“同行评审”（peer review），这现在已经成了金科玉律。

是的。这（同行评审）对世界上每一家资助机构都至关重要。我还没见过哪家不使用它。当然，除了我们自己的运作，我们不用它。但我们……我们找到了绕开它的方法。而传统的观点是，没有办法可以绕开。

他们认为没有办法。同行评审被认为是确保卓越研究的**唯一**途径。人们不停地这么说。但是我们已经（通过BP的风险研究和在UCOL的项目）证明了，还有另一条路。

**主持人：** 我猜，那些您向其提议的人，他们的回应是不是……他们不相信这（您的方法）可行，因为它没有经过同行评审？这是主要的症结（contention）吗？

**主持人：** 所以只是因为他们如此坚定地押注在同行评审是检验研究质量的唯一标准（acid test）上，他们就像……他们无法回头……他们就像被牢牢绑在了那艘船的桅杆上。

**Donald Braben：** 是的。

**主持人：** 好的。我知道至少在美国国家科学基金会（NSF），我记得好像是在您的书（第一版）最初出版后不久，大概在2008年、2009年。我读到过一项倡议，试图做更多……我想他们称之为“变革性研究”（transformational research）。那是NSF的倡议。

**Donald Braben：** 休斯（Hughes），你是说？

**主持人：** 是的，霍华德·休斯医学研究所（Howard Hughes Medical Institute）。没错。特别是他们的Janelia园区。据我了解，他们给人们五年的自由资金，真的就让他们去探索他们想探索的东西。

**Donald Braben：** 但他们……但我认为他们坚持让这些人去他们的中心实验室。

**主持人：** 是的。是的。

我的意思是，没有哪位科学家的重大发现是一开始就被欢呼所迎接的。爱因斯坦的发现……当他写了那篇著名的相对论论文时，《泰晤士报》称其为“**对常识的冒犯**”（an affront to common sense）。这就是《泰晤士报》当时的评价。

**主持人：** 所以，它确实是对常识的冒犯。是的。公平地说，它现在仍然是，对吧？就像你听到……“宇宙是弯曲的”，坦白说，这其实也不太符合我的常识。

**Donald Braben：** 宇宙是一个非常非常大的地方。

**主持人：** 是的，是一个非常怪异的地方，对吧？当你开始真正审视它的时候。

**Donald Braben：** 是的。你知道，哈勃常数，虽然具体数值还有争议，但它是每百万年一个微小的量。这是一个与我们现在所见的微小差异，如果事先不知道将很难被探测到。而现在你必须……你必须去寻找这个差异，这就是人们正在做的事情。

**主持人：** 是的。我一直在想一件事，不确定这的想法和“风险研究”是否契合。但我觉得，现在发生的情况是，人们在理论、实验、工程或开发方面变得如此专业化……以至于缺乏交流。而这些新领域的产生……部分是来自于人们与那些他们通常不会互动的人进行互动。

所以，我想问您……在您的“风险研究”社区的集体会议之外，您是否发现自己……会推动人们以他们原本不会主动采取的方式去互动？

**Donald Braben：** 不，我们从不那样“推”人。会议中产生的任何新互动，都完全源于科学家自己。我是说，我们……我们可能会对某个小组提出一两个建议。好吧，我确实……我们确实这么做过。

事实上，当时有研究蚂蚁的人，他们致力于“分布式智能”（distributed intelligence）领域。我碰巧知道爱丁堡大学有一个部门正在做这项工作，也就是“分布式计算”（distributed computing）。他们那里有一两位专家。于是我去找他们说：“你们是否有人有兴趣加入这个小组？”

他们确实派了一个人，名叫 Tuft。他去加入了蚂蚁小组，这是一次非常有成效的交流。但这（_指Braben主动撮合_）是非常罕见的。我的意思是，那次是发生了，但互动通常是反过来的（由科学家主动发起）。

**主持人：** 我明白了。

**Donald Braben：** 我不知道。也许……也许吧。你知道，我可能忘了，那是很久以前的事了。

**主持人：** 但是，我总是很有兴趣……试图改善我的心智模型，关于……每个人都会问的那个问题：“创意从何而来？” 对吧？我想知道“交叉授粉”（cross-pollination）对于创造新领域到底有多重要。

**主持人：** 实际上，我非常好奇运营BP“风险研究”项目的日常工作是什么样的。我在脑海中想象，您就像……飞遍世界，去见各种科学家。您看过《复仇者联盟》（Avengers）吗？我想象您就像尼克·弗瑞（Nick Fury），到处去找不同的超级英雄，然后说：“好了，我们要组建一支团队。” 是的。那（真实情况）是什么样的？

**Donald Braben：** 嗯，我告诉你，这是一个非常困难的问题。因为我刚到BP时，我只是孤零零的一个人，待在一个房间里。

当时，BP的研究总监，就是那个负责BP主要研究活动的人，他每年花费大约20亿英镑。他手下有2000人为他工作。而他，他认为我基本上是无害的。

但随着这个十年的推进，情况变了，因为我总是试图让BP的董事们参与到我们所做的事情中来。比如，我总是邀请他们参加我们的（Venture Research）会议。甚至BP的主席也来了，其他高级董事也来了。他们亲眼看到，我所谈论的并不是胡扯，而是非常严肃的事情。

因此，一种想法开始植入了一些董事的脑海中：那就是，Braben的“性价比”（bang for buck）比那位主流研究总监和他每年花费的20亿英镑要高得多。

而这种看法又反过来影响了那位研究总监对我的态度。他越来越把我看作是一个威胁，而不是一个机遇。最终，在1990年，他赢了，我们的项目被关闭了。

我（当时）在新西兰，接到了一个电话。在1990年3月8日，接到了来自新西兰的电话。是那位刚刚退休的研究总监Bob Malpas，他在任时给了我所需要的一切自由。他在1989年退休了，继任者是Basil Butler。

而Basil Butler……我不想说我对他有什么看法。他从新西兰打电话给我，他开口的第一句话就是：“喂，Don，‘风险研究’项目被关闭了。BP再也无法承受它对其资源的消耗了。”

我那时的花费是每年500万英镑，而BP的那些董事总经理们，如果一件事低于10亿，他们连床都懒得起。

**主持人**：是啊，真有意思，人们有时会非常在意那些相对很小的钱。

**Donald Braben**：人们倾向于从成本中看待价值。所以，如果一所大学采纳了风险研究——我希望他们会——他们也没什么光荣可言，因为这项目每年几乎不花钱。 即使你设立了寻找这些人的机制，比如伦敦大学学院（UCL），十年也只花了15万英镑，这个数字差不多是对的。我们谈论的是全世界一个世纪里大概500人。所以任何一所大学，十年里可能只会有一两个这样的人。 但是，如果能建立起这样的机制：人们可以带着他们的想法，去和大学里的高层人士交谈；而这些高层人士像我一样，已经放弃了自己的研究，转而从他人的发现中获得“替代的快乐”。 如果能找到这样的人，哪怕只有少数几所大学能做到，问题就解决了。 这就是我们现在努力的方向。我的书这周就该到了，Stripe 寄给我的50本样书一到，我就会把它们寄出去。恐怕我不是很乐观。

**主持人**：好吧。但我很乐观。也许我意识到这很傻，但我的理论是，如果你不乐观，你就注定会失败。不乐观会自我强化。如果你悲观，悲观就会自我实现。 所以我们需要保持乐观。我想，对于今天的大学来说，只要钱在那里，只要研究经费是从别的地方来的，而且不附带任何条件，那么大学对于有人在那里做研究是没意见的，对吧？

**Donald Braben**：是的。UCL 这样的大学，预算有十亿英镑，是所规模很大的大学。我们要求的只是一笔极小的钱，甚至这个数字都高估了，因为大多数年份的支出将是零。 只是需要能够偶尔调用，比如三年20万英镑。这对他们来说不是什么大的承诺。尤其是如果你能让他们期望，几年后，这些科学家将过渡到主流，并带回外部资金，就像我所做的那样。

**主持人**：我知道有几所大学确实会给新教授一两年的启动资金。

**Donald Braben**：没错。刚入职的人处于创造力最旺盛的时期。所以这对大学来说是很好的投资。但是，现在的学者必须向前看，思考这笔资金用完后会发生什么。他必须规划好自己的位置，以便能吸引到后续资金。 所以一两年的时间不足以做风险研究。这种方式有用，但程度非常有限。

**主持人**：是的，这种激励机制会反向传导。因为你会想：我未来需要申请经费，而要申请经费，我就需要有同行评审过的研究成果。所以我最好现在就开始做那些（能通过评审的）研究。 是的，这很有道理。 那么在最后，我想请问，除了阅读这本书和思考风险研究之外，您认为人们还应该思考哪些他们思考得不够的事情？

**Donald Braben**：我不认为你能这么说。如果你是一个风险研究者，或者一个萌芽中的风险研究者，你就会有一个想法，并且总想回到那个想法上。 所以，风险研究的创立是一个愉快的巧合，是志同道合者的相遇。我为那40个人提供了机会，我们在十年间支持他们去做他们想做的事。 但这是一种伙伴关系，是我们和他们之间的伙伴关系。我在 BP 内部承担着风险，必须解决各种问题。我必须做你说的所有那些事，比如环游世界，发表演讲。 因为我不认为我们可以登广告，在期刊上说“我们需要好点子”。我必须去大学，就风险研究和当今科学现状发表演讲，然后邀请人们提交提案，坐下来倾听他们带来了什么。 在每所大学，我一个下午可能会收到二三十份提案。他们中的大多数只是把你看作一个新的资金来源。 我们一直面临一个问题，即使是面对风险研究者，也要说服他们：虽然我们是 BP 风险研究部门，但我们对从地下采石油不感兴趣。这可能会帮助研究主管，但他不信任我们要做的事。 所以我必须找到一种方法，让我们的策略与研究主管的完全不同。我们没有直接竞争，但他确实视我为直接竞争，因为那些高层主管说：布拉本（我）取得的“性价比”比你的高。我没这么说，是他们说的。

**主持人**：您不公开发布申请征集，而是选择去演讲，然后和人们交谈。这背后的考虑是什么？我的直觉告诉我这很有道理，但我说不清为什么。

**Donald Braben**：风险研究者，诺贝尔奖得主，他们是非常特殊的一类人。他们不会回应所谓的“机会”。他们创造自己的机会，并且坚信自己对世界的独特看法最终会被证明是正确的。 幸运的是，对于我们帮助过的少数人，我们使他们能够做到这一点。但其他人则不得不根据现有的资金来调整自己的规划，并且必须一直这样做。 我认为，人们对世界的看法是在他们自身内部、在他们的思维过程中、在他们的创造精神中产生的。我们都拥有这种精神，它让我们能够做到这些。 比如爱因斯坦，他审视这个世界时，没有任何人的反馈。他不读文献，在他的“奇迹年”的三篇论文中没有引用任何文献。马克斯·普朗克是那些论文提交期刊的编辑，他必须决定是发表这些论文，还是按常规送去评审。他没有送审，他直接发表了。 这些论文招致了很多批评，《泰晤士报》称之为“对常识的冒犯”。但另外两篇关于电导率和布朗运动的论文，也都是重大的成果。 他做这些的时候没有和任何人讨论，除了偶尔从别人那里获得一些数学上的建议。

**主持人**：所以，回到筛选高质量申请者的策略上，假设就是：他们甚至根本不会去读那些刊登着征集申请广告的期刊，你必须真正走到他们面前去。

**Donald Braben**：是的。你必须创造一个环境，让他们愿意写信给你、联系你，或者打电话给你说：我想做这个。 我记得有一次接到一个“疯子”的电话，他说他有办法发射卫星，比任何人的方法都便宜得多。 我问：哦？你想怎么做？ 他说：我想建一座100英里高的大楼，然后从窗户把物体扔出去，它们马上就能进入轨道。 但我问他：你认为建造的极限是什么？一座100英里高的大楼需要什么样的地基？ 电话那头沉默了。因为他意识到地壳无法支撑。世界上最高的结构是珠穆朗玛峰，也就5英里高。你怎么去建造100英里高的东西？ 这不重要，只是我接到的一个电话的轶事。但这就是构成与我们初步接触的方式。我们所要求的只是，这个人打个电话，或者写一段话，说：这就是我想做的。

**主持人**：然后你们再从那里开始跟进。我非常喜欢这一点。这几乎与我所见的许多其他地方的做法完全相反。比如 DARPA 或贝尔实验室，他们的方式几乎是反过来的，他们总是主动去找人说：我们想让你来做一些了不起的事情。 这种在吸引人才方面的“推”与“拉”的动态，非常有趣。

**Donald Braben**：我最近和内阁办公室的一个人谈过这件事。他知道了我的书要出版，他说英国政府正考虑在英国设立 DARPA。 我认为这不是一个好主意。不是说对风险研究不好，它对其他事情有好处。 但是，DARPA 有点像风险投资，他们知道自己想要什么。而一个风险研究者在刚开始时，是无法指出他们的工作会带来什么具体好处的。因此他们会被取消申请资格。

**主持人**：是的，我认为您刚才做的区分非常重要。因为当人们说“科研体系坏掉了”时，我的假设是，他们实际上在描述至少两个完全不同的问题：一个问题是如何制造更多的“技术”；另一个问题是如何发现更多“新知识领域”。而风险研究瞄准的正是后者，与前者截然不同。

**Donald Braben**：新水果和新大陆。这就是我们专注的。全新的大陆，全新的水果。然后，从那些新水果中，又会产生新的“低垂的果实”。这就是我们努力在做的事。

**主持人**：我认为这次对话给我最大的收获之一，我想要再次强调的是，唐纳德断言：天才和疯子之间的界限，并不像我过去认为的那么模糊。并且，我们或许有可能通过真正关注一个人想法的“精确程度”来区分他们。我还在消化这一点，但这是个值得我们深思的重要问题。
