

# 1. “创世任务”执行路线图与关键时间轴

[Genesis Mission](https://genesis.energy.gov/)

![[Genesis Mission.png]]

创世任务（Genesis Mission）的执行时间表以行政命令签署日（2025年11月24日）为起点，设定了一系列具有侵略性的里程碑，以确保“美国科学与安全平台”（ASSP）的快速构建和运行。

| 里程碑                 | 时限 (从 2025年11月24日起)     | 核心目标与责任方                                                                                                 |
| :------------------ | :---------------------- | :------------------------------------------------------------------------------------------------------- |
| **国家挑战清单提交**        | **60天内** (约 2026年1月下旬)  | 能源部（DOE）部长必须向总统科学与技术助理（APST）提交 **至少20项** 具有国家重要性的科技挑战的详细清单。                                              |
| **计算资源识别**          | **90天内** (约 2026年2月下旬)  | DOE部长必须识别所有可用的联邦计算、存储和网络资源，包括DOE现场高性能计算（HPC）系统、安全的云端AI计算环境，以及来自**行业伙伴的资源**。                              |
| **初始数据集和模型资产识别**    | **120天内** (约 2026年3月下旬) | DOE部长必须识别一套初始的 **数据和模型资产**，包括进行**数字化、标准化、元数据和溯源追踪**。同时，制定一项计划，安全地纳入联邦资助研究、其他机构、学术机构和 **批准的私营部门伙伴** 的数据集。 |
| **机器人/自主设施审查**      | **240天内** (约 2026年7月下旬) | DOE部长必须审查DOE国家实验室及其他联邦研究设施中 **机器人实验室和生产设施** 的能力，以评估其进行 **AI导向的实验和制造**（包括自动化和AI增强型工作流程）的能力和所需标准。          |
| **初始运行能力 (IOC) 展示** | **270天内** (约 2026年8月下旬) | DOE部长必须寻求针对 **至少一项** 国家挑战，展示美国科学与安全平台（ASSP）的 **初始运行能力**。                                                 |
| **首次年度报告**          | **1年内** (约 2026年11月)    | DOE部长必须向总统提交年度报告（通过APST和OMB），描述平台的运营状态、科学成果（包括可衡量的科学进展、出版物和原型技术）以及**公私合作的范围和成果**。                        |

# 2. 参与机构、公司与核心领域总结

“创世任务”旨在调动全国的研发资源，包括联邦机构、学术界和私营部门的创新者。

### A. 领导机构与科研基础

| 机构/实体                        | 角色与职责                                                                     |
| :--------------------------- | :------------------------------------------------------------------------ |
| **美国能源部 (DOE)**              | 任务的 **执行主导机构**，负责在内部实施任务，整合17个国家实验室的资源，并将DOE的超级计算机和数据集整合到统一的平台中。          |
| **总统科学与技术助理 (APST)**         | **总体领导和协调** 机构，通过国家科学技术委员会（NSTC）确保跨部门（如NSF, NIH, NIST）的活动与任务目标保持一致。       |
| **能源部科学副部长 (Dr. Darío Gil)** | 被任命为 **“创世任务”主任**，负责领导该倡议的日常运营。                                           |
| **国家实验室** (17个)              | 任务的核心资源库，提供超级计算机、AI系统、量子系统以及40,000名科学家和工程师的专业知识。                          |
| **布鲁克海文国家实验室 (BNL)**         | 具体的国家实验室参与者，预计将在 **微电子学制造、电网运营和安全**，以及**发现科学**（包括“具身AI”和“安全AI”）等项目发挥关键作用。 |
| **橡树岭国家实验室 (ORNL)**          | 提到正在此地建设 **两个主权AI系统**。                                                    |
| **国家核安全管理局 (NNSA)**          | 将利用AI、量子计算和先进数据分析来**加强威慑力量**，并确保美国 **核武库的安全性和可靠性**。                       |
